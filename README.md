# interscape
AI - based internship recommendation system based on students performance and skills.
A lightweight AI-based recommendation system that suggests the most relevant internships
to candidates under the PM Internship Scheme.

## Problem Statement
Many first-generation learners struggle to identify suitable internships from large listings.
This project provides 3–5 personalized internship recommendations based on skills,
education, interests, and location.

## Features
- Profile-based internship recommendations
- Rule-based / ML-light scoring engine
- Mobile-friendly UI
- Simple and accessible design
- Backend APIs using FastAPI

## Tech Stack
- Frontend: React.js, TypeScript, TailwindCSS
- Backend: FastAPI, Python
- Database: PostgreSQL
- Auth: JWT

## How It Works
1. Candidate enters basic details
2. System calculates compatibility score
3. Top 3–5 internships are shown as cards

## Impact
Helps underprivileged and first-generation learners discover relevant internship opportunities.
